Hello, welcome to my webpage!

I built this webpage from scratch using html and css. Html stands for Hypertext-Markup-Language. Hypertext means "text with links in it." A markup language is a programming language used to make text do more than just sit on a page: it can turn text into images, links, tables, lists, and much more. CSS stands for cascading style sheets. CSS is what makes the webpage stylish. Its like the skin and makeup and html is like the skeletal structure.
